# Bedrock服务器多设备同账号登录解决方案

## 问题说明

Bedrock服务器默认情况下，**同一个Microsoft账号不能同时在多台设备上登录**。这是因为Xbox Live认证系统的限制。

## 解决方案

### 方案1：使用不同的Microsoft账号（推荐）

为每台设备创建独立的Microsoft账号：
1. 电脑使用账号A
2. 手机使用账号B
3. 两个账号都可以同时登录服务器

**优点：**
- ✅ 完全支持，无任何限制
- ✅ 最安全和稳定的方式
- ✅ 符合Minecraft官方建议

### 方案2：使用本地世界（LAN）

如果设备在同一局域网内：
1. 在一台设备上创建本地世界
2. 其他设备通过局域网加入
3. 本地世界允许同一账号多设备登录

**限制：**
- ⚠️ 只能在局域网内使用
- ⚠️ 需要设备在同一网络

### 方案3：修改服务器配置（实验性）

**注意：** 此方法可能不稳定，不推荐用于生产环境。

Bedrock服务器可能支持通过修改配置来允许这种情况，但需要：
1. 设置 `online-mode=false`（会降低安全性）
2. 或者使用特殊的服务器版本

**风险：**
- ⚠️ 可能违反Minecraft服务条款
- ⚠️ 安全性降低
- ⚠️ 可能导致其他问题

## 当前服务器配置

您的服务器当前配置：
- `online-mode=true` - 要求Xbox Live认证
- `allow-list=false` - 未启用白名单

## 建议

**最佳实践：**
1. 为每台设备使用不同的Microsoft账号
2. 如果必须使用同一账号，考虑使用本地世界（LAN）
3. 避免修改 `online-mode` 为 `false`，这会降低安全性

## 关于玩家名称后缀

您提到的"在账号名称后面添加[x]"的方法：
- 这是某些第三方服务器软件的特性
- 官方Bedrock服务器**不支持**此功能
- 如果使用第三方服务器软件（如BDSX或其他），可能支持此功能

## 如何检查当前登录的玩家

可以通过服务器日志或使用命令查看当前在线玩家：
```bash
# 查看服务器日志
tail -f /home/ubuntu/bedrock-server/Dedicated_Server.txt | grep "Player"
```
